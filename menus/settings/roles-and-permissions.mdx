The Roles & Permissions page shows all roles available in your organization and their access levels.

<Info>Custom roles with granular permission controls are coming soon. You'll be able to create roles with specific permissions tailored to your organization's needs.</Info>

## System Roles

Beni includes three built-in system roles. These cannot be edited or deleted.

| Role | Type | Description |
|------|------|-------------|
| **Owner** | System · Protected | Full organization control with protected access. Only one Owner per organization. |
| **Admin** | System | Organization administration excluding billing ownership actions. |
| **Member** | System | Operational access for knowledge base and conversation workflows. |

The **Owner** role is marked **Protected** — it cannot be modified or reassigned through normal settings.

## Custom Roles

The ability to create custom roles with granular permission controls is **coming soon**. This will allow you to define exactly what each role can access and manage within your workspace.
